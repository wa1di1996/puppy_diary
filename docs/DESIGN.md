## 一、产品定位

**一句话：** 多人协作的宠物日常记录 App。

**核心场景：** 情侣/家庭共同养宠，各自记录喂食、遛狗、便便、洗澡、美容、健康等日常，共享同一只宠物的档案。

**目标用户：** 养宠家庭（2-4 人共同照顾 1-2 只狗）。

## 二、用户角色与权限模型

### 2.1 角色定义

| 角色 | 权限 |
|------|------|
| **主人（owner）** | 创建宠物的人。完整权限：编辑档案、记录日常、生成分享邀请、管理成员（移除）、删除宠物 |
| **共享成员（guardian）** | 通过邀请码加入。可查看宠物数据、记录日常、查看成员列表。**不可**删除宠物/管理成员/生成邀请 |

### 2.2 权限矩阵

| 操作 | 主人 | 共享成员 |
|------|:--:|:--:|
| 查看宠物信息 | ✅ | ✅ |
| 记录日常（喂食/遛狗/便便/营养品） | ✅ | ✅ |
| 健康管理（洗澡/美容/体重/健康） | ✅ | ✅ |
| 查看记录列表 | ✅ | ✅ |
| 查看品种信息 | ✅ | ✅ |
| 查看费用计算器 | ✅ | ✅ |
| 编辑档案 | ✅ | ❌ |
| 生成分享邀请码 | ✅ | ❌ |
| 移除共享成员 | ✅ | ❌ |
| 删除宠物 | ✅ | ❌ |

### 2.3 分享流程

```
主人 → 点击"分享" → 选择长期/限时 → 生成8位邀请码
                                          ↓
共享成员 → "接受邀请" → 输入邀请码 → 查询 → 选择身份(长期) → 加入
```

## 六、功能模块明细

### 6.1 日常记录（4 类型）

| 类型 | 字段 | 说明 |
|------|------|------|
| 喂食 | 食物类型(狗粮/零食/自制) + 喂食量(g) + 关联狗粮 + 备注 | 可关联已购狗粮品牌 |
| 遛狗 | 时长(分钟) + 时段(早/上/下/晚) + 心情(5级) + 备注 | 心情：😢不想走→🤩疯跑 |
| 便便 | 评分(1-5) + 备注 | 合并了原独立粑粑评分页 |
| 营养品 | 名称 + 费用(元,可选) + 备注 | 鱼油、关节宝等 |

### 6.2 健康管理（4 类型）

| 类型 | 字段 | 说明 |
|------|------|------|
| 洗澡 | 已完成标记 + 费用(可选) + 备注 | |
| 美容 | 已完成标记 + 费用(可选) + 备注 | |
| 体重 | 体重(kg) + 备注 | 输入后即时对比品种标准区间 |
| 健康 | 子类型 + 专属字段 | 疫苗/体内驱虫/体外驱虫/体检 |

### 6.3 费用计算器

- 按月度汇总 5 类费用（狗粮/营养品/洗澡/美容/医疗）
- 对比上月，显示趋势箭头
- 数据自动从记录和狗粮中提取

### 6.4 品种信息

- 6 种狗数据：西高地、泰迪、金毛、拉布拉多、法斗、柯基
- 展示：标准体重/身高/寿命、营养需求、运动需求、性格特点
- 基于档案体重计算个性化喂食量

## 八、产品经理审查

### 8.1 做得好的

1. **权限模型清晰** — owner/guardian 二元角色，简单有效
2. **分享流程简洁** — 8 位邀请码，两步加入
3. **记录分类合理** — 日常(高频) vs 健康管理(低频)分离
4. **费用计算器** — 自动汇总，宠物主关心的月度开销一目了然
5. **便便评分可视化** — emoji + 5 级评分，直观

### 8.2 需要补充的功能

| 优先级 | 功能 | 理由 |
|--------|------|------|
| P0 | **疫苗/驱虫提醒** | 养狗刚需，错过疫苗可能影响健康证办理 |
| P0 | **洗澡/美容周期自定义** | 目前只有"标记完成"，没有自动提醒下次 |
| P1 | **多宠物切换时保留数据上下文** | 切换宠物后首页数据应实时刷新 |
| P1 | **遛狗排便关联** | 遛狗时可直接标记是否排便 + 评分，减少操作 |
| P2 | **照片墙/相册** | 便便照片、头像等集中浏览 |
| P2 | **数据导出** | 更换 App 或给兽医看时的数据迁移 |
| P3 | **多语言** | 当前硬编码中文 |
| P3 | **暗色模式** | 晚上查看不刺眼 |

### 8.3 养宠人士视角的问题

| 问题 | 影响 | 建议 |
|------|------|------|
| 没有「饮水量」记录 | 饮水量变化是肾病早期信号 | 加一条简单的饮水量记录 |
| 没有「异常行为」记录 | 如呕吐、挠痒、跛行等 | 健康管理加"异常"类型 |
| 疫苗/驱虫无提醒 | 忘记驱虫会导致寄生虫 | P0 提醒功能 |
| 狗粮消耗不追踪 | 不知道一包狗粮能吃多久 | 狗粮关联喂食记录自动计算剩余 |
| 找不到附近宠物医院 | 紧急情况下的刚需 | 可加地图入口（P3） |

## 十一、已知限制

1. 图片以 base64 存 SQLite，大量图片会膨胀数据库
2. 无 HTTPS（备案后配置）
3. 无自动备份（可加 cron job 定期备份 data.db）
4. 品种数据硬编码 6 种，不支持用户自定义品种
