# 汪星日记（Puppy Diary）— 产品评审 & 重构指引

> 本文档基于2026-05-11对 http://47.115.207.89/ 的完整产品体验测试，经Owner逐条确认，用于指导下一轮vibe coding重构。

---

## 一、产品定位

宠物日常记录App，核心场景：多人共同养宠，各自记录宠物的喂食、遛狗、便便等日常，共享宠物档案。

目标用户：情侣/家庭共同养宠（如一对情侣养柯基+金毛）。

---

## 二、Owner确认事项（已定论，不可再议）

| 确认项 | 决定 | 理由 |
|--------|------|------|
| "记录一下"按钮 | **保留**，是主入口 | 核心功能放首页最醒目位置合理 |
| 底部"记录日常"菜单 | **删除** | 与"记录一下"重复 |
| "编辑档案"按钮 | **删除** | 与"狗狗档案"菜单重复 |
| 底部"体重记录"菜单 | **删除** | 标准体重在主页已有，记录日常-体重保留 |
| 底部"粑粑评分"菜单 | **删除** | 合并到记录日常-便便 |
| 驱虫 | **拆分**为体内驱虫/体外驱虫 | 两种驱虫药完全不同 |
| 驱虫+疫苗 | **添加提醒功能** | 养狗刚需 |
| 就医/用药 | **不添加** | 属于意外场景，非日常需求 |
| 狗粮记录 | **定位为购买记录**（品牌/价格/日期） | 方便比价，与喂食记录功能不同 |
| 洗澡和美容 | **保持独立，不合并** | 频率不同，需各自设置周期提醒 |
| 营养品 | **单独开一栏**，不从喂食中选 | 与喂食是不同维度 |
| 喂食食量 | **保留克数** | 养狗人按克重喂食 |
| 配色方案 | **不调整** | 琥珀色主色调已确认 |
| 遛狗心情 | **添加情绪图标+描述** | 😢不想走/😐勉强走/🐶正常/😊精力充沛/🤩疯跑 |
| 喂食次数统计 | **不需要** | Owner确认 |
| 遛狗次数统计 | **不需要** | Owner确认 |
| 费用计算器 | **新增页面** | 统计各类费用（狗粮/营养品/洗澡/美容/医疗） |
| 日历计划 | **支持添加/调整/预览计划** | 洗澡美容体检等可提前规划 |
| 全局TabBar | **不加** | 与子页面二级路径冲突 |

## 四、记录日常 — 4种类型完整表单定义

> 所有类型共用：日期(date) + 类型专属字段 + 备注(textarea) + 快捷日期(今天/昨天/前天) + 保存

```
- 日期 date（默认今天）
- 食类型 select：狗粮 / 零食 / 自制狗饭
- 喂食量 number（placeholder "例如: 100"，单位g）
- 关联狗粮 select：（当食类型=狗粮时，可选择狗粮记录中已有的品牌）
- 备注 textarea
- 快捷日期 + 保存
```

### 4.2 遛狗

```
- 日期 date（默认今天）
- 时段 select：早晨 / 上午 / 下午 / 晚上（默认根据当前时间）
- 时长 number（placeholder "例如: 30"，单位分钟）
- 心情 select/图标：😢不想走 / 😐勉强走 / 🐶正常 / 😊精力充沛 / 🤩疯跑
- 是否排便 checkbox（勾选后展开便便评分子表单）
  ├── 评分：💥拉稀 / 🍵偏软 / ✅正常 / 🌟很好 / 🏆完美
  └── 照片 file（可选）
- 备注 textarea
- 快捷日期 + 保存

### 4.3 便便

```

- 评分 select/图标：💥拉稀 / 🍵偏软 / ✅正常 / 🌟很好 / 🏆完美
- 照片 file（可选）
- 备注 textarea
